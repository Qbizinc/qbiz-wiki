Welcome to Ruddr!

  1. You should have received an email invitation to join Ruddr. Click on Join Workspace

2. Once you accept the invitation you will be asked to set up a password. When you log into the Qbiz workspace you will see your personal dashboard page (which will be empty in the beginning)

3. If you are on a billable project, you should be assigned to it already and you should be able to see it by going to Home → My Projects on the left navigation bar. 

4.To submit Timesheets, go to Time → Time Entries on the left navigation bar

5. Select the Week view

6. Click on Add Time Row button

6.1 If you are submitting project time (billable or non-billable) select Project Time type and you should see your client project or the Internal Tasks project to select from. As an example:

6.2 If you are submitting time off, select Time Off type and choose from the available options

7. Once you are done with the weekly time entries, click on Submit for Approval at the bottom right corner

 

A couple things to keep in mind:

  • Timesheets are due at the end of each week
